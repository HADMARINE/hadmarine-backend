# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 4.2.x   | :white_check_mark: |
| < 4.2   | :x:                |

## Reporting a Vulnerability

There are three methods to solve a vulnerability

1. contact to contact@hadmarine.com
2. Open an issue of this project
3. Make a pull request
